№3 Напишите функцию min(a,b), которая возвращает меньшее из чисел a и b.

Решение:
function min(a, b) {
  if (a < b) {
    return a;
  } else {
    return b;
  }
}
или
function min(a, b) {
  return a < b ? a : b;
}

№5 Используя цикл while, напишите код, в котором выводятся числа от 0 до 3, 

Решение:

while (i < 3) { 
  alert( i );
  i++;
}


№6 Напишите программу, в которой переменной number присвойте любое число.
Используя if...else сравнивайте это число с 0 и выводите соответствующие сообщения: 
'Число больше 0'; 'Число меньше нуля'; 'Число равно 0'

Решение:
let number = 10;

if (number > 0) {
  console.log('Число больше 0');
} else if (number < 0) {
  console.log( 'Число меньше нуля' );
} else {
  console.log( 'Число равно 0' );
}


№7 Используя switch перепишите код

  console.log('Вы ввели число 0');
}

if (number === 1) {
  console.log('Вы ввели число 1');
}

if (number === 2 || number === 3) {
 console.log('Вы ввели число 2, а может и 3');
}

Решение:
const number = 2;

switch (number) {
  case 0:
    console.log('Вы ввели число 0');
    break;

  case 1:
    console.log('Вы ввели число 1');
    break;

  case 2:
  case 3:
    console.log('Вы ввели число 2, а может и 3');
    break;
}
